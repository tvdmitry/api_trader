Гибкие настройки
торговых стратегий
Широкие возможности настройки условий покупки и продажи по индикаторам рынка позволяют эффективно работать как на медвежьем так и на бычьем рынке.
Полнофункциональный
график
Больше не нужно пользоваться сторонними сервисами для получения наглядной
информации о торговле. Наш график отображает все необходимые индикаторы рынка, сетку ордеров, сделки, историю торговли.
функциональная сетка
ордеров со встроенным калькулятором
Сетка ордеров с широким диапазоном настроек. Встроенный калькулятор с возможностью расчета перекрытия цены, процента отскока и требуемого депозита.
Сервис анализа
и подбора торговых пар
Инструмент позволяет проводить анализ всех торговых пар на бирже и выбирать наиболее подходящие для торговли в текущий период времени
Модуль симуляции торговли
Позволяет тестировать выбранную стратегию торговли на исторических данных без риска потерять деньги.
Функционал автоматического
подбора пары
Позволяет автоматически выбирать наиболее подходящую пару для торговли исходя из заданных параметров из списков заранее созданных пользователем
Развернутая статистика результатов торговли
Подробная статистика произведенных операций на бирже, с точной информацией по каждому торговому циклу и сделке.
Настройка и запуск сервиса займет 15 минут.
